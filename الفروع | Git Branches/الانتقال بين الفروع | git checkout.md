# الانتقال بين الفروع 

يتم الانتقال بين الفروع من خلال الامر :

<div dir="ltr">

```
$ git checkout new-branch
```

<div dir="rtl">

مثال توضيحي :


<div dir="ltr">

```
$ git branch
* master
  new-branch
$ git checkout new-branch
Switched to branch 'new-branch'
$ git branch
  master
* new-branch
```
