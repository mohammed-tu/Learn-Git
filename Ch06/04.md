# إنشاء فرع والإنتقال له 

يتم إنشاء فرع جديد والإنتقال له بخطوة واحدة مختصرة عن طريق الامر التالي :


<div dir="ltr">

```
$ git checkout -d new-branch-1
```
<div dir="rtl">

مثال توضيحي :

<div dir="ltr">

```
$ git branch
  master
* new-branch
$ git checkout -d new-branch-1
fatal: git checkout: --detach does not take a path argument 'new-branch-1'
$ git branch
  master
* new-branch
```
