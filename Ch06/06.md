# حذف فرع 

يتم حذف فرع تم إنشائه سابقا عبر الامر التالي :

<div dir="ltr">

```
$ git branch -d branch
```

<div dir="rtl">

مثال توضيحي :

<div dir="ltr">

```
$ git branch
* dev-new-branch
  master
  new-branch
$ git branch -d new-branch
Deleted branch new-branch (was 9837eb2).
$ git branch
* dev-new-branch
  master
```

<div dir="rtl">

**ملاحظة هامة :**
لا يمكن حذف الفرع الفعال
