# إنشاء الفروع وعرضها 

يتم إنشاء الفرع من خلال الأمر التالي :

<div dir="ltr">
  
  ```
  $ git branch new-branch
  ```
<div dir="rtl">

يتم عرض الفروع من خلال الامر التالي :

<div dir="ltr">
  
  ```
  $ git branch 
  ```

<div dir="rtl">

مثال توضيحي :

<div dir="ltr">
  
  ```
$ git branch
* master
$ git branch new-branch
$ git branch
* master
  new-branch
  ```
