#  إنشاء وعرض العلامات | tags

 العلامات هي نقاط في تاريخ المشروع يشار إليها 
 وعادة ما يتم التعبير عنها بالاصداات للمشروع 

يتم إنشاء العلامة بالامر التالي :
 

<div dir="ltr">

```
$ git tag v1.0.0
```

<div dir="rtl">
  
يتم عرض العلامة بالامر التالي :
 

<div dir="ltr">

```
$ git tag 
```
