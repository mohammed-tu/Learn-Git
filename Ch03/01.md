  # البداية مع Git
  
  ### الأمر `git init`
  
  هو أمر لإنشاء ملف ال .git في المشروع المراد تتبع النسخ فيه 
  وهو ملف مخفي حيث أنه يبدأ بعلامة النقطة . 
  
  **طريقة إستخدامه :** 
  
  <div dir="ltr">
  
  ```
$ mkdir git-project
$ cd git-project/
$ ls
$ git init
Initialized empty Git repository in C:/Users/temx3/Desktop/tuwaiq-git/Giit/git-project/.git/
$ ls
$ ls -a
./  ../  .git/

  ```
